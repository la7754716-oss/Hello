# 英语词汇学习影响因素的实证研究

## 项目简介

本研究探讨了移动应用使用频率、英语四级学习程度和高考语文成绩对中国大学生英语词汇水平的影响。采用UVLT（University Vocabulary Levels Test）作为测量工具，通过多元线性回归分析揭示了不同因素在基础词汇和高级词汇层次上的差异性作用。

## 研究概述

- **样本量：** 75名非英语专业大学生
- **研究设计：** 横断面调查研究
- **测量工具：** UVLT Level 1000和Level 3000
- **分析方法：** 多元线性回归与AICc模型选择（R语言AICcmodavg包）

## 主要发现

### 模型1：基础词汇（UVLT 1000；score1）最优模型
- **APP使用频率（Frequency）** 显著正向影响（β=0.531, p=0.00116**）
- **四级学习程度（Degree）** 显著正向影响（β=0.347, p=0.02258*）
- **模型拟合：** R²=0.263, F(2,72)=12.85, p<0.001

### 模型2：较高阶词汇（UVLT 3000；score2）最优模型
- **APP使用频率（Frequency）** 显著正向影响（β=0.394, p=0.01117*）
- **高考语文成绩（Grade）** 显著正向影响（β=0.118, p=0.00817**）
- **模型拟合：** R²=0.263, F(2,72)=12.85, p<0.001

## 核心结论

1. **移动学习工具的普遍有效性：** APP使用频率对不同难度词汇均有稳定的正向影响
2. **学习投入的针对性效应：** 考试导向学习主要促进基础词汇习得
3. **母语能力的层次性迁移：** 跨语言认知能力在高级词汇学习中更为重要
4. **词汇习得的多层次机制：** 不同难度词汇需要采取差异化教学策略

## 实践启示

### 对教师
- 整合移动学习工具到教学体系
- 采取差异化词汇教学策略
- 适当利用学生母语资源

### 对学习者
- 坚持使用英语学习APP
- 平衡考试准备与能力发展
- 发挥母语分析能力优势

## 研究局限性

- 横截面设计限制因果推断
- 自我报告测量可能存在偏差
- 样本来自单一高校
- 模型解释力有待提升（约27-28%）

## 未来研究方向

- 纵向追踪研究验证因果关系
- 探讨中介和调节效应
- 增加更多词汇层次测试
- 结合质性研究方法
- 设计实验干预研究
