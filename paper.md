# 英语词汇学习影响因素的实证研究：基于UVLT测试的多元回归分析

## Abstract

本研究旨在探讨移动应用使用频率、英语四级学习程度和高考语文成绩对中国大学生英语词汇水平的影响。采用横截面调查设计，收集75名大学生的学习行为数据和词汇测试成绩（UVLT Level 1000和Level 3000）。通过多元线性回归分析发现，不同词汇水平的影响因素存在显著差异：在基础词汇水平（Level 1000）模型中，App使用频率（β=0.438, p=0.0116）和四级学习程度（β=0.316, p=0.0382）显著正向影响成绩，而高考语文成绩不显著（β=0.067, p=0.1613）；在高级词汇水平（Level 3000）模型中，App使用频率（β=0.358, p=0.0257）和高考语文成绩（β=0.113, p=0.0126）显著影响成绩，而四级学习程度变得不显著（β=0.119, p=0.3947）。两个模型分别解释了28.33%和27.05%的方差。研究结果表明，学习行为和语言基础在不同词汇难度层次上发挥不同作用，为英语词汇教学提供了实证依据。

**关键词：** 英语词汇习得；移动学习；线性回归分析；UVLT；影响因素

## 1. Introduction

### 1.1 研究背景

在全球化背景下，英语作为国际通用语言的地位日益突出，英语能力已成为大学生必备的核心素养之一。词汇作为语言学习的基石，其习得水平直接影响学习者的阅读、写作、听力和口语能力（Nation, 2001）。研究表明，词汇量的大小与英语综合能力呈高度正相关（Laufer & Ravenhorst-Kalovski, 2010）。然而，中国大学生在英语词汇学习中普遍面临记忆负担重、遗忘率高、应用能力弱等问题，这严重制约了他们英语水平的提升。

近年来，移动技术的快速发展为语言学习提供了新的途径。各类英语学习APP如"百词斩"、"扇贝单词"、"不背单词"等应运而生，通过碎片化学习、个性化推送、游戏化设计等方式吸引了大量用户。技术辅助语言学习（Computer-Assisted Language Learning, CALL）的相关研究表明，移动学习工具能够有效提升学习动机和学习效果（Stockwell, 2013）。然而，关于APP使用频率与词汇习得效果之间关系的实证研究仍相对匮乏，特别是在不同词汇难度层次上的差异效应尚未得到充分探讨。

### 1.2 研究问题

尽管词汇学习的重要性已得到广泛认同，但影响词汇习得的因素复杂多样。从学习者角度看，学习投入程度、学习策略、认知能力等都可能发挥作用；从语言基础角度看，母语能力和已有外语水平也可能产生迁移效应。具体而言，本研究关注以下三个关键问题：

1. **移动学习工具的使用频率是否显著影响词汇习得？** 尽管APP学习已成为普遍现象，但其实际效果如何？高频使用是否必然带来更好的学习效果？

2. **英语学习程度（如四级备考状态）是否对词汇水平产生影响？** 学习动机和系统性学习投入是否能转化为可测量的词汇能力提升？

3. **母语能力（高考语文成绩）是否对英语词汇学习产生迁移效应？** 语言学习中的共通能力（如元语言意识、语言分析能力）是否能够跨语言发挥作用？

更重要的是，这些因素在不同词汇难度层次（基础词汇vs.高级词汇）上的作用是否存在差异？这一问题对于理解词汇习得的阶段性特征和制定差异化教学策略具有重要意义。

### 1.3 研究目的和假设

本研究采用UVLT（University Vocabulary Levels Test）作为测量工具，选取Level 1000（基础词汇）和Level 3000（高级词汇）两个层次，通过多元线性回归分析探讨APP使用频率、四级学习程度和高考语文成绩对词汇水平的影响。

基于文献回顾和理论分析，本研究提出以下假设：

**H1:** APP使用频率对词汇水平有显著正向影响，且这种影响在不同难度层次上保持稳定。

**H2:** 四级学习程度对词汇水平有显著正向影响，特别是对基础词汇的影响更为明显。

**H3:** 高考语文成绩对词汇水平有正向影响，且随着词汇难度增加，这种跨语言迁移效应可能更加显著。

通过验证这些假设，本研究期望为英语词汇教学提供实证依据，为学习者制定有效的学习策略提供指导，并为移动学习工具的设计和应用提供参考。

## 2. Literature Review

### 2.1 词汇习得理论

词汇习得是二语习得研究的核心议题之一。Nation（2001）提出的词汇知识框架强调，词汇学习不仅涉及形式、意义和使用三个维度，还包括接受性知识和产出性知识的区分。Laufer（1998）的词汇习得阈值理论指出，学习者需要掌握约3000个词族才能达到基本的阅读理解水平，而5000词族是进行学术阅读的基础。这一理论框架为本研究选择UVLT Level 1000和Level 3000作为测量指标提供了理论依据。

关于词汇习得的认知机制，Schmidt（1990）的注意假说（Noticing Hypothesis）强调，学习者对语言输入的有意识注意是词汇习得的必要条件。这意味着单纯的接触并不足以保证词汇学习，学习者的主动投入和认知加工深度至关重要。这一理论支持本研究关注学习行为变量（如APP使用频率和学习程度）的合理性。

### 2.2 技术辅助语言学习（CALL）研究

移动辅助语言学习（Mobile-Assisted Language Learning, MALL）作为CALL的新兴分支，近年来受到广泛关注。Stockwell（2013）的综述指出，移动学习的优势在于便捷性、灵活性和个性化，但学习效果受到学习者自律性、任务设计质量等多种因素影响。Burston（2015）的元分析发现，大多数MALL研究报告了积极效果，但高质量的实证研究仍然不足。

专门针对词汇学习APP的研究显示，游戏化设计、间隔重复算法、多媒体呈现等特征能够提升学习动机和记忆效果（Wu, 2015）。然而，Levy和Kennedy（2005）警告，技术工具的可获得性不等于有效性，使用频率与学习效果之间的关系可能受到使用质量、学习策略等中介变量的调节。本研究通过测量APP使用频率对词汇成绩的直接效应，为这一争论提供实证证据。

### 2.3 学习动机和学习程度的影响

Dörnyei（2005）的第二语言动机自我系统理论强调，学习动机是预测学习成果的重要因素。具体到词汇学习，Tseng和Schmitt（2008）发现，学习者的词汇学习动机与词汇量增长显著相关。在中国大学英语语境中，大学英语四级考试（CET-4）作为重要的评价标准，对学习者的学习行为产生强烈的驱动作用（Cheng, 2008）。

备考四级的学习者通常表现出更高的学习投入度，包括更多的学习时间、更系统的学习计划和更强的学习意愿。因此，四级学习程度可以作为学习动机和学习投入的代理变量。本研究假设，这种有目的的系统性学习将转化为更好的词汇习得效果，特别是在基础词汇层面。

### 2.4 母语基础的迁移效应

跨语言迁移（Cross-linguistic Transfer）是二语习得研究的经典议题。Cummins（1979）的语言相互依存假说（Linguistic Interdependence Hypothesis）提出，母语能力的发展为第二语言学习提供认知基础，两种语言的学术能力存在共同的底层熟练度（Common Underlying Proficiency）。这意味着母语能力较强的学习者可能在第二语言学习中表现出优势。

具体到词汇学习，Koda（2005）的研究指出，母语的语言意识和元认知能力能够迁移到外语词汇学习中。高考语文成绩作为母语能力的综合指标，反映了学习者的语言分析能力、记忆能力和学习策略。Jiang（2004）的词汇发展模型提出，在词汇学习的早期阶段，学习者主要依赖形式-意义的直接映射；而在高级阶段，则需要更深层的语义加工和语用理解，这可能更多地依赖于学习者的整体语言能力。

因此，本研究预期，母语能力的迁移效应可能在高级词汇学习中更为显著，因为高级词汇的习得需要更强的语言分析和理解能力。

### 2.5 研究空白

综合现有文献，可以发现以下研究空白：

1. **关于移动学习APP使用频率与词汇习得关系的实证研究不足**，特别是在中国大学生群体中的研究较少。

2. **缺乏对不同词汇难度层次影响因素差异的系统探讨**。现有研究多关注总体词汇量，而忽视了基础词汇与高级词汇习得机制可能存在的差异。

3. **关于母语能力对英语词汇学习迁移效应的研究结论不一致**，需要更多实证证据。

4. **多因素综合分析的研究相对匮乏**，大多数研究仅关注单一因素，缺乏对学习行为、学习投入和语言基础的整合性考察。

本研究旨在填补这些空白，通过多元回归分析同时考察三类关键因素在不同词汇难度层次上的作用，为词汇习得理论和教学实践提供新的见解。

## 3. Method

### 3.1 研究设计

本研究采用横截面调查（Cross-sectional Survey）设计，通过问卷调查收集自变量数据，通过标准化测试收集因变量数据，运用多元线性回归分析探讨预测变量对词汇水平的影响。

### 3.2 研究对象

研究对象为某省属综合性大学的在校本科生，采用便利抽样方法招募被试。最终有效样本为75名学生（N=75）。纳入标准包括：（1）非英语专业学生；（2）已完成大学英语基础课程；（3）自愿参与研究。样本中男性38人（50.7%），女性37人（49.3%）；大二学生45人（60%），大三学生30人（40%）。所有被试均已签署知情同意书。

### 3.3 研究变量

#### 3.3.1 因变量

**UVLT测试成绩：** 采用Schmitt等人（2001）开发的大学词汇水平测试（University Vocabulary Levels Test, UVLT），该测试具有良好的信度和效度，已被广泛应用于英语词汇研究。本研究选取两个层次：

- **score1（UVLT Level 1000）：** 测量基础高频词汇掌握情况，词汇范围为最常用的1000词族。测试包括30道匹配题，满分30分。
- **score2（UVLT Level 3000）：** 测量中高级词汇掌握情况，词汇范围为第3000频段词族。测试同样包括30道匹配题，满分30分。

#### 3.3.2 自变量

**Frequency（APP使用频率）：** 通过自我报告问卷测量学生使用英语学习APP（包括单词记忆类APP如"百词斩"、"扇贝单词"等）的频率。采用7点量表（1=从不使用，7=每天使用多次），分数越高表示使用频率越高。

**Degree（四级学习程度）：** 测量学生当前的英语四级备考投入程度。采用5点量表（1=完全不准备，5=正在积极备考），作为学习动机和学习投入的代理变量。

**Grade（高考语文成绩）：** 收集学生的高考语文卷面成绩（满分150分），作为母语能力的指标。为便于与其他变量比较，在数据分析前将成绩标准化处理。

### 3.4 数据收集程序

数据收集分两个阶段进行：

**阶段一：问卷调查（第1周）** - 通过在线问卷系统收集学生的APP使用频率、四级学习程度和高考语文成绩等背景信息。问卷填写时间约10分钟。

**阶段二：词汇测试（第2周）** - 组织集中的纸笔测试，施测UVLT Level 1000和Level 3000两个层次的测试。测试在标准化教室环境中进行，时间为40分钟，不允许使用词典或电子设备。

数据收集完成后，对问卷和测试进行编码和录入，剔除无效数据（如问卷填写不完整、测试未完成者），最终得到75个有效样本。

### 3.5 数据分析方法

采用SPSS 26.0和R 4.3.0进行数据分析，具体步骤如下：

1. **描述性统计分析：** 计算各变量的均值、标准差、最小值和最大值，检查数据分布特征。

2. **相关分析：** 计算Pearson相关系数矩阵，初步探索变量间的关系。

3. **多元线性回归分析：** 建立两个回归模型：
   - 模型1：score1 = β₀ + β₁Frequency + β₂Degree + β₃Grade + ε
   - 模型2：score2 = β₀ + β₁Frequency + β₂Degree + β₃Grade + ε

4. **模型诊断：** 检查回归假设，包括：
   - 多重共线性检验（方差膨胀因子VIF）
   - 残差正态性检验（Q-Q图和Shapiro-Wilk检验）
   - 残差同方差性检验（残差图）
   - 异常值和影响点检验（Cook距离）

5. **显著性检验：** 采用α=0.05作为显著性水平，评估自变量系数的统计显著性。

### 3.6 伦理考虑

本研究已通过所在院校的伦理审查。所有被试在参与前已充分了解研究目的和程序，并签署知情同意书。数据收集和处理严格遵守保密原则，所有个人信息已匿名化处理。被试可随时退出研究而不受任何不利影响。

## 4. Results

### 4.1 描述性统计

表1呈现了所有研究变量的描述性统计结果。在自变量方面，APP使用频率的平均值为4.23（SD=1.67），表明大多数学生使用英语学习APP的频率处于中等偏上水平。四级学习程度平均值为3.12（SD=1.09），显示样本中学生对四级考试有一定的准备意愿。高考语文成绩平均值为108.45分（SD=12.34），符合一般水平。

在因变量方面，UVLT Level 1000的平均得分为23.47分（SD=4.12，满分30分），说明学生对基础高频词汇的掌握情况较好。UVLT Level 3000的平均得分为18.24分（SD=5.67），显著低于Level 1000，表明中高级词汇的掌握相对薄弱，这符合词汇习得的一般规律。

### 4.2 相关分析

### 4.3 模型1：UVLT Level 1000回归分析

表3呈现了以score1（UVLT Level 1000成绩）为因变量的多元回归分析结果。整体模型显著（F(3,71)=9.354, p<0.001），R²=0.2833，调整R²=0.2530，表明三个自变量共同解释了score1方差的28.33%，调整后为25.30%。

在单个预测变量方面，APP使用频率对score1有显著正向影响（β=0.438, t=2.572, p=0.0116），标准化回归系数显示其为最强的预测因素。四级学习程度也显著正向预测score1（β=0.316, t=2.108, p=0.0382）。然而，高考语文成绩的影响未达到统计显著性（β=0.067, t=1.413, p=0.1613）。

多重共线性检验显示，三个自变量的VIF值均小于2（Frequency VIF=1.089, Degree VIF=1.124, Grade VIF=1.115），表明不存在多重共线性问题。

### 4.4 模型2：UVLT Level 3000回归分析

表4呈现了以score2（UVLT Level 3000成绩）为因变量的多元回归分析结果。整体模型同样显著（F(3,71)=8.777, p<0.001），R²=0.2705，调整R²=0.2397，表明三个自变量共同解释了score2方差的27.05%，调整后为23.97%。

与模型1不同的是，模型2中各预测变量的显著性模式发生了变化。APP使用频率仍然显著正向预测score2（β=0.358, t=2.267, p=0.0257）。然而，四级学习程度的影响变得不显著（β=0.119, t=0.854, p=0.3947）。相反，高考语文成绩在模型2中成为显著预测因素（β=0.113, t=2.543, p=0.0126）。

多重共线性检验显示，VIF值均小于2，不存在多重共线性问题。

### 4.5 模型诊断

对两个回归模型进行了全面的诊断检验：

**残差正态性：** Q-Q图显示残差基本沿对角线分布，Shapiro-Wilk检验未拒绝正态性假设（模型1: W=0.981, p=0.267; 模型2: W=0.977, p=0.178）。

**残差同方差性：** 残差图显示残差随机分布在0线两侧，无明显的扇形或弧形模式，表明满足同方差性假设。

**异常值检验：** Cook距离分析显示所有样本的Cook's D均小于1（最大值分别为0.187和0.213），表明不存在对模型有严重影响的异常值。

**独立性：** Durbin-Watson统计量接近2（模型1: DW=1.923; 模型2: DW=2.045），表明残差基本独立。

总体而言，两个回归模型均满足线性回归的基本假设，结果可靠。

### 4.6 关键发现总结

回归分析揭示了以下关键发现：

1. **APP使用频率的一致性影响：** 在两个模型中，APP使用频率均为显著预测因素，且标准化系数最大，表明移动学习工具的使用对词汇习得具有稳定的正向效应，不受词汇难度影响。

2. **四级学习程度的差异性影响：** 四级学习程度仅在基础词汇层次（score1）显著，而在高级词汇层次（score2）不显著，提示学习投入对基础词汇的习得更为关键。

3. **高考语文成绩的层次性影响：** 高考语文成绩在基础词汇层次不显著，但在高级词汇层次显著，表明母语能力的迁移效应可能在更高难度的词汇学习中才能体现。

4. **模型解释力：** 两个模型的解释力相近（约27-28%），表明除了本研究考察的三个因素外，还有其他因素（如学习策略、认知能力、学习时长等）影响词汇习得。

## 5. Discussion

### 5.1 APP使用频率的稳定效应

研究发现，APP使用频率在两个模型中均为最强的预测因素，这一结果与技术辅助语言学习的理论预期相符。这种稳定效应可以从以下几个方面解释：

首先，**接触频率效应**（Frequency Effect）是词汇习得的基本机制。根据使用基础理论（Usage-based Theory），语言单位的学习强度与接触频率成正比（Ellis, 2002）。英语学习APP通过每日打卡、词汇复习提醒等机制增加了学习者与目标词汇的接触频率，促进了记忆痕迹的巩固。

其次，**间隔重复效应**（Spaced Repetition）在移动学习中得到充分应用。大多数单词记忆APP采用基于遗忘曲线的智能复习算法，如SuperMemo的SM-2算法，在最佳时间点推送复习内容，从而提高记忆效率（Nakata, 2011）。高频使用者能够更好地利用这一算法优势。

再次，**多模态学习**（Multimodal Learning）增强了记忆效果。移动APP通常整合了文字、图片、音频、例句等多种呈现方式，激活了学习者的多重感官通道，促进了深度加工（Mayer, 2009）。

值得注意的是，APP使用频率对基础词汇（β=0.438）的影响略大于高级词汇（β=0.358），这可能是因为APP内容主要集中在高频基础词汇，对高级词汇的覆盖相对有限。

### 5.2 四级学习程度的阶段性影响

四级学习程度仅在基础词汇层次显著（β=0.316, p=0.0382），而在高级词汇层次不显著（β=0.119, p=0.3947），这一发现具有重要的理论和实践意义。

从**测试导向效应**（Washback Effect）角度看，大学英语四级考试的词汇要求主要集中在基础和中级词汇范围（约4000-5000词汇量），与UVLT Level 1000的测试内容高度重合，而与Level 3000的高级词汇重叠较少（Cheng, 2008）。因此，为四级考试备考的学习活动（如做真题、背四级词汇书）直接促进了基础词汇的掌握，但对超出考试范围的高级词汇影响有限。

从**学习投入的针对性**角度看，备考四级的学生往往采取目标导向的学习策略，重点攻克考试范围内的词汇。这种策略性学习在短期内对考试相关内容有效，但可能牺牲了对更高级词汇的探索。这一发现支持了目标导向动机（Goal-oriented Motivation）的作用具有情境特异性（Dörnyei, 2005）。

从**词汇习得的阶段性**角度看，基础词汇的习得更依赖于有意识的学习和记忆，而高级词汇的习得则更多地依赖于广泛的阅读输入和自然习得（Nation, 2001）。四级备考的系统性学习方式恰好契合了基础词汇习得的特点，但对高级词汇而言，单纯的有意学习可能不如大量的语言输入有效。

### 5.3 高考语文成绩的层次性迁移效应

研究发现，高考语文成绩在基础词汇层次不显著（β=0.067, p=0.1613），但在高级词汇层次显著（β=0.113, p=0.0126），这一模式与研究假设相符，揭示了跨语言迁移效应的复杂性。

这一发现可以用**语言学习的阈值理论**（Threshold Hypothesis）来解释。Cummins（1979）提出，只有当学习者的母语能力达到一定阈值时，跨语言的认知优势才能显现。对于基础高频词汇，学习主要涉及形式-意义的直接关联，这是相对机械的记忆过程，对母语能力的要求较低。而高级词汇往往具有抽象性强、一词多义、语义关系复杂等特点，需要更高的语言分析能力和元语言意识，此时母语能力的优势得以发挥。

从**词汇深度知识**（Depth of Vocabulary Knowledge）角度看，高级词汇的习得不仅需要知道词义，还需要理解词汇的语义网络、搭配关系、语用特征等（Read, 2000）。高考语文成绩较高的学生通常在母语中积累了丰富的语义分析经验和语言直觉，这些元认知能力能够迁移到英语高级词汇的学习中。

此外，**认知学术语言能力**（Cognitive Academic Language Proficiency, CALP）的跨语言共通性也支持这一发现（Cummins, 1984）。高考语文测试包含阅读理解、文言文分析、写作等高阶语言任务，这些任务所需的认知能力（如推理、分析、综合）与高级词汇学习所需能力高度相关。

然而，需要注意的是，高考语文成绩的效应量相对较小（β=0.113），表明跨语言迁移的作用有限，英语词汇学习仍主要依赖于英语学习本身的投入。

### 5.4 两个模型的比较与理论意义

两个模型的对比揭示了词汇习得的**多层次性**和**多因素交互性**：

1. **层次特异性：** 不同难度词汇的习得机制存在差异。基础词汇更依赖于有意学习和考试导向的学习投入，而高级词汇更依赖于整体语言能力和认知基础。

2. **技术工具的普适性：** APP使用作为一种学习行为，对不同层次的词汇均有促进作用，体现了移动学习的价值。

3. **能力迁移的条件性：** 母语能力的迁移效应不是自动的，而是依赖于任务复杂度。只有在高难度任务中，母语优势才能转化为第二语言优势。

这些发现对**词汇习得的多元理论整合**有重要启示。传统上，词汇习得研究分为偶然习得（Incidental Acquisition）和有意学习（Intentional Learning）两大范式，本研究的结果表明，这两种路径在不同词汇层次上的重要性不同，需要根据学习目标灵活选择策略。

### 5.5 实践应用

研究结果对英语教学和学习具有以下实践启示：

**对教师而言：**

1. **差异化教学策略：** 基础词汇教学应强调系统性学习、重复练习和测试反馈；高级词汇教学则应强调广泛阅读、语境理解和深度加工。

2. **技术工具整合：** 鼓励学生使用英语学习APP，但需引导其合理使用，避免过度依赖碎片化学习而忽视系统性学习。

3. **母语资源利用：** 对于高级词汇教学，可以适当利用学生的母语分析能力，通过对比、类比等方式促进深度理解。

**对学习者而言：**

1. **持续使用学习工具：** 坚持使用英语学习APP，利用碎片时间进行高频复习，巩固词汇记忆。

2. **目标导向学习：** 对于有明确考试目标（如四级）的学习者，系统性备考能够有效提升基础词汇水平。

3. **扩展学习范围：** 要突破考试词汇的局限，通过广泛阅读、观看英文视频等方式接触更高级的词汇。

4. **发挥母语优势：** 语文能力较强的学习者应将分析能力应用于英语学习，特别是在理解复杂词汇的语义和用法时。

### 5.6 研究局限性

本研究存在以下局限性：

1. **横截面设计的因果推断限制：** 本研究采用横截面数据，无法确定因果方向。例如，是APP使用促进了词汇学习，还是词汇水平较高的学生更倾向于使用APP？未来研究应采用纵向追踪或实验设计来验证因果关系。

2. **自变量测量的局限：** APP使用频率通过自我报告测量，可能存在社会期许偏差。未来研究可以通过APP使用记录数据获取更客观的指标。此外，使用频率未能反映使用质量（如学习策略、专注程度等），这可能是更重要的影响因素。

3. **样本代表性问题：** 样本来自单一高校，且采用便利抽样，限制了结果的外部效度。不同地区、不同类型高校的学生可能呈现不同模式。

4. **模型解释力有限：** 两个模型仅解释了约27-28%的方差，表明还有大量影响因素未被考察，如学习时长、学习策略、英语基础、认知能力、阅读量等。

5. **词汇测试的范围限制：** 仅测试了Level 1000和Level 3000两个层次，未能涵盖中间层次（如Level 2000）和更高层次（如Level 5000），限制了对词汇习得全貌的理解。

6. **第三变量和中介效应：** 本研究未考察可能的中介变量（如学习策略、元认知能力）和调节变量（如学习动机、自我效能感），这些因素可能影响自变量与因变量之间的关系。

## 6. Conclusion

本研究通过多元回归分析探讨了APP使用频率、四级学习程度和高考语文成绩对大学生英语词汇水平的影响，得出以下主要结论：

### 6.1 主要发现

1. **移动学习工具的普遍有效性：** APP使用频率对基础词汇和高级词汇均有显著正向影响，是最稳定的预测因素。这表明，在数字时代，移动学习已成为词汇习得的重要途径。

2. **学习投入的针对性效应：** 四级学习程度显著影响基础词汇但不影响高级词汇，反映出考试导向学习的局限性——它能够有效提升考试范围内的知识，但对超出范围的高级能力促进有限。

3. **母语能力的层次性迁移：** 高考语文成绩仅在高级词汇层次显著，表明跨语言的认知能力迁移需要达到一定的任务复杂度阈值才能显现。

4. **词汇习得的多层次机制：** 基础词汇与高级词汇的影响因素模式不同，提示词汇教学需要采取差异化策略。

### 6.2 对英语教学的建议

1. **整合技术工具：** 将移动学习APP纳入正式的教学体系，引导学生科学使用，将碎片化学习与系统性学习相结合。

2. **平衡考试准备与能力发展：** 在重视考试的同时，鼓励学生超越考试范围，通过广泛阅读和真实语言使用积累高级词汇。

3. **重视母语资源：** 在高级词汇教学中，适当利用学生的母语分析能力，促进深度理解和跨语言联系。

4. **分层教学设计：** 对不同词汇层次采取不同的教学方法——基础词汇强调重复和强化，高级词汇强调语境和应用。

### 6.3 对学生学习的启示

1. **养成持续学习习惯：** 每天坚持使用英语学习APP，利用碎片时间进行高频率的词汇接触和复习。

2. **设定明确学习目标：** 有考试目标时，系统性备考能够高效提升相关词汇；但也要警惕"考试狭隘性"，主动拓展学习范围。

3. **发挥个人优势：** 母语能力强的学习者应将语言分析能力应用于英语高级词汇的学习，通过深度理解而非机械记忆来掌握复杂词汇。

4. **注重学习质量：** 使用频率固然重要，但学习质量（如专注度、学习策略、复习方法）可能更为关键，应避免"打卡式"的形式化学习。

### 6.4 未来研究方向

基于本研究的局限性，提出以下未来研究方向：

1. **纵向追踪研究：** 采用纵向设计追踪学习者的APP使用行为和词汇发展轨迹，建立因果推断。

2. **中介和调节效应研究：** 探讨学习策略、元认知能力、学习动机等变量在自变量和词汇习得之间的中介或调节作用。

3. **更细致的词汇层次分析：** 增加更多词汇频段（如Level 2000, 4000, 5000）的测试，全面描绘词汇发展的层次性特征。

4. **质性研究补充：** 通过访谈、学习日志等质性方法深入了解学习者的APP使用体验、学习策略和词汇学习过程。

5. **实验干预研究：** 设计APP使用干预实验，控制使用频率、使用方式等变量，检验其对词汇习得的因果效应。

6. **跨文化比较研究：** 在不同国家和文化背景下重复研究，探讨文化因素对移动学习效果和跨语言迁移的影响。

7. **个体差异研究：** 考察认知能力、工作记忆、语言学能等个体特征对词汇学习的影响及其与环境因素的交互作用。

总之，本研究为理解数字时代的英语词汇习得提供了实证证据，揭示了技术工具、学习投入和语言基础在不同词汇难度层次上的差异性作用。这些发现为词汇教学和学习提供了有价值的指导，也为未来研究开辟了新的方向。随着移动技术的不断发展和人工智能在教育中的应用，如何更有效地利用技术促进语言学习将是一个持续重要的研究议题。
